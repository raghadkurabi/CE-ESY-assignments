# Assignment 1 - Circular Buffer

## الفكرة (Idea)
تنفيذ مخزن دائري (Circular Buffer) بلغة C للتعامل مع حالتي الامتلاء والفراغ، والتأكد من أن البيانات تُقرأ وتُطبع بشكل صحيح.

## خطوات البرنامج (Program Steps)
1. إدخال الاسم من المستخدم.
2. إضافة النص "CE-ESY" إلى الاسم.
3. كتابة النص داخل المخزن الدائري حرفًا بحرف.
4. قراءة النص من المخزن وطباعة النتيجة.
5. التأكد أن المخزن فارغ بعد القراءة.
